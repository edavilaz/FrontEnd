El proyecto desarrollado consiste en un sitio web sobre reseñas de películas, el cual permite encontrar la próxima película que se quiere ver o crear una lista de favoritas. El sitio web muestra un listado de las últimas películas organizadas por fecha, no existe la posibilidad de aplicar filtros pero se espera agregarlo en la futura evolucion de la actividad al igual que una función de búsqueda. Al hacer clic en una película del listado, se muestra su detalle con información relevante como la sinopsis, actores, fotos y la opción de agregar una reseña, la cual se almacena de manera temporal a travez de javascript manipulando los elementos del DOM.

El proyecto se desarrolló siguiendo las especificaciones indicadas en las indicaciones básicas. Se utilizó el Framework CSS Bootstrap y se implementó el sistema de grillas para organizar los elementos del sitio. Además, se adaptó el diseño a las tres resoluciones planteadas, usando las clases correspondientes en BOOSTRAP version 5: <576px móviles (. col-), Tablet ≥576px (. col-sm-), Desktop ≥768px (col-md-). Se siguieron buenas prácticas de maquetación de código XHTML.

El proyecto también integra el API de The Movie Database, permitiendo la obtención de información actualizada y relevante de las películas.
